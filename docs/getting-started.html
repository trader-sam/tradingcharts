Getting started

OpenCharts owns one responsive Canvas inside a host element. Give that host a measurable height, load your OHLC data, and release the chart when the host is permanently removed.

LIVE EXAMPLE · OHLC

Install

npm install opencharts

During local development, import from ./src/index.ts. Published consumers import from opencharts.

Create a chart

import { createChart, type Bar } from 'opencharts';

const host = document.querySelector<HTMLElement>('#chart')!;
host.style.height = '420px';

const chart = createChart(host, {
  background: '#0b1020',
  crosshair: true,
  yAxis: 'right',
});

const bars: Bar[] = [{ time, open, high, low, close, volume }];
chart.setData(bars);
The host controls chart size. OpenCharts observes host resizes; do not set a canvas width or height yourself.

Update a live chart

chart.update(nextBar);       // append newer data or replace an active timestamp
chart.updateMany(batch);     // merge a feed burst and redraw once
chart.setVolumeVisible(true);
chart.fitContent();          // return to the visible data extent

Use the same timestamp while a candle is still forming. Supply the next interval timestamp only when that bar closes.

Lifecycle

const chart = createChart(host);
// ...use chart
chart.destroy(); // disconnects listeners and observers when the host is removed
