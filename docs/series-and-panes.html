Series and panes

Indicators are application-owned data. OpenCharts renders the points you provide as a line, histogram, area, or step-area without imposing an indicator catalog.

LIVE EXAMPLE · AREA SERIES IN A LOWER PANE

Create an independently scaled pane

chart.addPane({ id: 'signal', title: 'Momentum', height: 0.28 });

const signal = chart.createSeries({
  id: 'momentum', pane: 'signal', type: 'area',
  color: '#a78bfa', fillColor: '#3c2c69',
  negativeFillColor: '#703647'
});

signal.setData([{ time, value: 12.4 }, { time: nextTime, value: -4.1 }]);
signal.update({ time: nextTime, value: -3.8 });

Create the pane before assigning a series to it. A pane shares horizontal navigation with the main chart but owns its Y range, pan state, and resize divider.

Choose a renderer and scale

chart.addSeries({ id: 'ema', type: 'line', color: '#fbbf24', scale: 'price' });
chart.addSeries({ id: 'volume-score', type: 'histogram', scale: 'independent' });
chart.addSeries({ id: 'depth', type: 'step-area', fillColor: '#1f5d47' });
  • scale: 'price' shares main OHLC coordinates.
  • scale: 'independent' derives a dedicated range from the series.
  • fillColor controls area fill; negativeFillColor changes the fill below zero.

Use custom-only charts

const chart = createChart(host, {
  primarySeriesVisible: false,
  ohlcTooltip: false,
  xAxis: { type: 'numeric', formatter: (x) => `$${x}` }
});
chart.createSeries({ id: 'payoff', type: 'area' }).setData(points);

For series-only charts, time remains a stable point identity; optional x supplies numeric or ordinal axis values. No placeholder OHLC bars are necessary.
