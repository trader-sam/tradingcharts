Axes and interaction

Time is the default X domain. Numeric and ordinal domains are equally first-class, using caller-provided values and stable label anchors as you zoom.

LIVE EXAMPLE · NUMERIC X AXIS

Configure an X domain

chart.setXAxis({ type: 'time' }); // default: milliseconds since epoch

chart.setXAxis({
  type: 'numeric',
  formatter: (value) => `$${Number(value).toFixed(0)}`
});
chart.setData([{ time: 0, x: 95, open, high, low, close }]);

chart.setXAxis({ type: 'ordinal', formatter: (value) => `Contract ${value}` });

Time axes keep calendar anchors stable: January uses a year label, month starts remain fixed, then days and intraday labels appear as zoom increases. Numeric axes use their actual continuous coordinates, preserve irregular spacing, and generate readable value ticks from the visible range.

Pointer controls

  • Drag the plot to pan freely in X. On numeric axes, this moves the continuous numeric viewport; price remains fit to visible data until its Y axis is unlocked.
  • Wheel the plot to zoom around the cursor. Numeric axes retain the cursor's exact coordinate; time-series X zoom keeps the final visible bar anchored.
  • Drag or scroll a Y axis to unlock and scale that pane vertically. Axis cursors show the available direction.
  • Drag a lower-pane divider to resize a pane; drag inside that pane to pan its own Y range.
  • Hover the plot for crosshair, price, and X-axis labels.
  • Keyboard: focus a chart, then use Left/Right to pan, +/− to zoom, Home to fit content, and Escape to cancel an active drawing.

Fit and reset

chart.fitContent();
chart.setType('line');
chart.setDrawingTool('trendline');

Synchronize charts and load history

const unsubscribe = chart.subscribeVisibleRangeChange((range) => {
  if (range && range.from < 20) loadEarlierBars();
});

chart.setVisibleLogicalRange({ from: 120, to: 180 });
chart.setVisibleRange({ from: Date.UTC(2026, 0, 1), to: Date.UTC(2026, 0, 31) });
chart.scrollToRealTime();
unsubscribe();

Logical and time ranges are intentionally available for time and ordinal charts. A continuous numeric X axis has its own coordinate viewport, so these methods return null rather than pretending bar indexes are numeric positions.

Charts use touchAction: 'pan-y' by default: vertical touch movement remains available to the surrounding page, while horizontal chart gestures work normally. Use interaction to disable wheel or drag navigation, or choose a different touch-action policy for a dedicated chart surface.
